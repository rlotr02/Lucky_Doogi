## 🔑 핵심 구현
- Canvas API를 활용하여 마우스 및 터치 드래그로 화면을 긁어내는 스크래치 인터랙션 구현
- HTML img 태그의 onLoad 이벤트를 활용하여 에셋 로드가 완료된 시점에 드로잉을 시작하도록 동기화
- 다시하기 기능 동작 시 상태 리셋 순서 정합성을 보정하여 이전 결과 이미지가 잔상으로 튀는 화면 깜빡임 차단
- Styled-components 미디어 쿼리 기반의 웹 및 모바일 디바이스별 스케일 최적화 레이아웃 구현
